# Wddmzjyzs -反JY软件
## [By GWL](https://g255x.github.io)
------------------------功能介绍------------------------

        TaskMgr   循环启动任务管理器(输入0)
        Kill      杀死极域电子教室(输入1)
        TaskList  进程列表(输入2)
        TaskKill  杀死某进程(输入3)
        TaskPause 暂停/恢复某进程(输入4)
        RemoveREG 解除注册表禁用项(输入5)
        RemoveUsb 解除U盘禁用(输入6)
        Help      帮助列表(输入7)

        按下C键结束极域电子教室
        按下B建结束控制软件
        任务管理器按ALT键退出
        按下ESC+Ctrl关闭程序
